## 第二天：非线性规划（50分）

### 上午（3h）— KKT + 凸分析

**目标：KKT条件烂熟于心**

1. **凸分析基础（1h）**
   - 第3章：凸集定义 → 凸函数定义 → 一阶/二阶判别
   - 局部极小=全局极小（凸规划核心定理）
   - 凸函数判别：Hessian半正定 ↔ 凸函数

2. **KKT条件（1.5h）**
   - 第7章重点页：FJ条件 → KKT条件 → Lagrange函数写法
   - **KKT条件写法的标准步骤**：
     1. 写出 Lagrange 函数 L(x,λ,μ)
     2. ∇_x L = 0（梯度条件）
     3. 互补松弛：μ_i g_i(x) = 0
     4. 原始可行性 + 对偶可行性
   - 手做真题Q6（FJ → KKT → 对偶）

3. **非线性对偶（30min）**
   - 弱对偶定理 → Slater约束规格 → 强对偶定理
   - 鞍点与KKT的关系

### 下午（3h）— 拟牛顿法 + 可行方向法

**目标：DFP/BFGS 能完整写出一轮迭代**

1. **拟牛顿法（1.5h）**
   - 第10章 81-118页：拟牛顿条件 → DFP公式 → BFGS公式
   - **DFP算法步骤**（必考！）：
     1. 给定 x₀, H₀（通常=I）
     2. 计算梯度 g₀，搜索方向 d₀ = -H₀g₀
     3. 一维搜索求步长 λ₀，得 x₁
     4. 计算 s₀ = x₁-x₀, y₀ = g₁-g₀
     5. DFP校正：H₁ = H₀ + s₀s₀ᵀ/s₀ᵀy₀ - (H₀y₀)(H₀y₀)ᵀ/y₀ᵀH₀y₀
     6. 重复
   - 手做真题Q4（DFP完整两轮迭代）

2. **可行方向法（1.5h）**
   - 第12章：Zoutendijk → Rosen梯度投影
   - **Rosen梯度投影步骤**（真题Q5）：
     1. 识别积极约束，构造矩阵 M
     2. 计算投影矩阵 P = I - Mᵀ(MMᵀ)⁻¹M
     3. 方向 d = -P∇f(x)
     4. 一维搜索求步长
   - 手做真题Q5

### 晚上（2h）— 罚函数法 + 共轭梯度

1. **罚函数法（1h）**
   - 第13章：外点法（惩罚因子→∞） vs 内点法（障碍因子→0）
   - 辅助函数的构造方式
   - 乘子法（增广Lagrange）的基本思想

2. **共轭梯度法（1h）**

   - FR共轭梯度法步骤 + β因子计算公式
   - 与拟牛顿法的区别：不存储矩阵，只需向量运算

## 风险提示

- **单纯形表迭代错误**：每一步更新后检查检验数和目标值
- **DFP公式记混**：BFGS是对B校正（不是H），DFP是对H校正
- **KKT忘记互补松弛**：μᵢgᵢ(x)=0 是最容易漏的条件
- **对偶单纯形方向反了**：单纯形法是保持原可行求对偶可行；对偶单纯形是保持对偶可行求原可行
