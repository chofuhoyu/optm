\subsection{6.1 无约束极值条件——起点}

无约束优化 $\min f(x)$ 的最优性条件是我们最熟悉的出发点。
PPT第7.3页回顾了这三个经典结论，是一切有约束理论的基石。

\medskip
\refslide{7最优性条件}{3}{7.1}
\medskip

\begin{defbox}
\textbf{无约束极值条件（定理 7.1.2--7.1.4）}

设 $f$ 适当光滑。
\begin{itemize}
  \item \textbf{一阶必要条件}：若 $x^*$ 是局部极小点且 $f$ 在 $x^*$ 可微，则 $\nabla f(x^*) = 0$。
  \item \textbf{二阶必要条件}：若 $f$ 在 $x^*$ 二次可微，则 $\nabla f(x^*)=0$ 且 $\nabla^2 f(x^*)$ 半正定。
  \item \textbf{二阶充分条件}：若 $\nabla f(x^*)=0$ 且 $\nabla^2 f(x^*)$ 正定，则 $x^*$ 是严格局部极小点。
\end{itemize}
\end{defbox}

加上约束后，最优解可能落在约束边界上，此时 $\nabla f(x^*) \neq 0$ 是常态——梯度不需要消失，只要沿\emph{可行方向}无法进一步下降即可。我们需要新工具来描述这种“受约束的静止”。

% ============================================================
\subsection{6.2 等式约束与 Lagrange 乘子法}

先从最简单的情形入手：\textbf{仅有等式约束}。这是理解一切乘子法的入口。

\begin{defbox}
\textbf{定理 6.1（Lagrange 乘子法，等式约束）}

考虑 $\min f(x)$，s.t.\ $h_i(x)=0\ (i=1,\dots,m)$。定义 Lagrange 函数
$L(x,\mu)=f(x)+\sum_{i=1}^m \mu_i h_i(x)$。

若 $x^*$ 是局部极小点且 $\nabla h_i(x^*)$ 线性无关（即 LICQ 成立），则存在唯一的乘子 $\mu_1^*,\dots,\mu_m^*$，使得：
\[
\nabla_x L(x^*,\mu^*) = \nabla f(x^*) + \sum_{i=1}^m \mu_i^*\nabla h_i(x^*) = 0
\]
\end{defbox}

\textbf{几何含义}：$-\nabla f(x^*)$ 落在约束法向量张成的子空间中。换句话说，目标函数梯度在约束切空间上的投影为零——沿任何可行方向移动，目标函数的一阶变化率都是零。

\textbf{【自编例题1】具体数字演示 Lagrange 乘子法}

\begin{notebox}
\textbf{例1：} $\min\ x_1^2 + x_2^2$，s.t.\ $x_1 + x_2 = 1$。\\
目标：到原点的最小距离平方，约束：直线 $x_1+x_2=1$。几何直觉：原点至直线的垂足 $(\frac12,\frac12)$ 应是最优解。
\end{notebox}

\textbf{解（Lagrange 乘子法）：}
构造 $L(x,\mu)=x_1^2+x_2^2+\mu(x_1+x_2-1)$。
\[
\begin{aligned}
\frac{\partial L}{\partial x_1} &= 2x_1 + \mu = 0 \;\Rightarrow\; x_1 = -\frac{\mu}{2} \\[2pt]
\frac{\partial L}{\partial x_2} &= 2x_2 + \mu = 0 \;\Rightarrow\; x_2 = -\frac{\mu}{2} \\[2pt]
\frac{\partial L}{\partial \mu} &= x_1 + x_2 - 1 = 0 \;\Rightarrow\; -\mu = 1 \;\Rightarrow\; \mu = -1
\end{aligned}
\]
得 $x^*=(\frac12,\frac12)^T$，$f^*=\frac12$。验证与几何直觉一致。

\textbf{解（FJ 条件验证，理解 $\lambda_0$ 的角色）：}
这里只有一个等式约束，FJ 条件为：存在不全为零的 $\lambda_0\ge 0$ 和 $\mu$，使得
\[
\lambda_0\begin{pmatrix}2x_1\\2x_2\end{pmatrix}
+ \mu\begin{pmatrix}1\\1\end{pmatrix} = \begin{pmatrix}0\\0\end{pmatrix},\quad x_1+x_2=1
\]
\textbf{情形 $\lambda_0=0$：} $\mu(1,1)^T=(0,0)^T \Rightarrow \mu=0$，全部乘子为零，不满足“不全为零”的要求。此情形无解。\\
\textbf{情形 $\lambda_0\neq 0$：} 归一化 $\lambda_0=1$，得 $2x_1+\mu=0,\; 2x_2+\mu=0,\; x_1+x_2=1$。与 Lagrange 法完全一致，得 $x^*=(\frac12,\frac12)^T$，$\mu=-1$。

\begin{notebox}
\textbf{关键观察：} 在此例中 $\lambda_0=0$ 的情形被“不全为零”条件排除，因为我们只有一个等式约束且其梯度 $\nabla h=(1,1)^T\neq 0$（非退化）。当约束梯度非退化时，FJ 与 KKT 等价。
\end{notebox}

\subsection{6.3 一般约束问题与 FJ 条件}

现在加入不等式约束，考虑一般形式的约束优化问题：
\[
\begin{aligned}
\min\ & f(x) \\
\text{s.t.}\ & g_i(x) \le 0 \quad (i=1,\dots,m) \\
& h_j(x) = 0 \quad (j=1,\dots,p)
\end{aligned}
\]

PPT第7.17页给出了 FJ 一阶最优性条件的形式——这是最一般、不依赖任何约束规格的必要条件。

\medskip
\refslide{7最优性条件}{17}{7.9}
\medskip

\begin{defbox}
\textbf{定理 6.2（Fritz John 条件，1948）}

设 $x^*$ 是上述问题的局部极小点，$f,g_i,h_j$ 在 $x^*$ 附近连续可微。则存在不全为零的乘子 $\lambda_0,\lambda_1,\dots,\lambda_m,\mu_1,\dots,\mu_p$，满足：
\[
\boxed{\lambda_0 \nabla f(x^*) + \sum_{i=1}^m \lambda_i \nabla g_i(x^*) + \sum_{j=1}^p \mu_j \nabla h_j(x^*) = 0}
\]
\[
\lambda_i \ge 0\ \ (i=0,1,\dots,m)
\]
\[
\lambda_i g_i(x^*) = 0\ \ (i=1,\dots,m)
\]
（$\mu_j$ 无符号约束，因为等式约束没有”方向”限制。）
\end{defbox}

\medskip

\textbf{FJ 条件从何而来？——从几何直觉到代数方程}

FJ 条件不是从天而降的。其几何直觉是：在局部极小点 $x^*$ 处，\emph{不存在可行下降方向}——即沿任何可行方向 $d$ 移动，$f$ 都不能下降。等价地说，负梯度 $-\nabla f(x^*)$（最速下降方向）与所有可行方向都形成钝角或直角：$-\nabla f(x^*)^T d \le 0$。

对积极不等式约束 $g_i(x^*) = 0$，可行方向必须满足 $\nabla g_i(x^*)^T d \le 0$（不能指向 $g_i>0$ 的方向）。对等式约束 $h_j(x^*) = 0$，可行方向必须满足 $\nabla h_j(x^*)^T d = 0$（必须沿切平面）。把这些方向约束与”无下降方向”的条件放在一起，由\emph{Farkas 引理}（或择一定理）可知：$-\nabla f(x^*)$ 必定落在积极约束梯度所生成的锥中。翻译成代数语言就是——存在不全为零的非负乘子 $\lambda_0,\lambda_i$ 和自由乘子 $\mu_j$，使得梯度线性组合为零：
\[
\lambda_0 \nabla f(x^*) + \sum_{i\in\mathcal{A}} \lambda_i \nabla g_i(x^*) + \sum_{j=1}^p \mu_j \nabla h_j(x^*) = 0
\]
其中 $\lambda_0 \ge 0$ 乘在目标梯度上，$\lambda_i \ge 0$ 乘在不等式约束梯度上（不等式只限制一侧方向，乘子非负），$\mu_j$ 乘在等式约束梯度上（等式限制两侧，乘子可正可负）。加上互补松弛 $\lambda_i g_i(x^*) = 0$（非积极的约束不参与），就得到了完整的 FJ 条件。

\textbf{注意 $\lambda_0$ 的特殊性：} 上述推导中，如果目标梯度本身就落在约束梯度锥中，会出现一种退化情况——目标函数在 $x^*$ 处的梯度信息被约束梯度”吞没”，$\lambda_0=0$。这正是下一小节要讨论的问题。

\medskip

\subsubsection{为什么 $\lambda_0$ 可以为零？——FJ 条件的”软肋”}

FJ 条件允许 $\lambda_0 = 0$ 是它的\emph{设计缺陷}，也是它的\emph{通用性代价}。当 $\lambda_0=0$ 时：
\[
\sum_{i=1}^m \lambda_i \nabla g_i(x^*) + \sum_{j=1}^p \mu_j \nabla h_j(x^*) = 0
\]
目标函数的梯度 \textbf{完全消失}。FJ 条件退化为一组仅涉及约束梯度的线性依赖关系——它不携带任何关于 $f(x)$ 的信息。

\textbf{几何情景：} 当某个约束的梯度在 $x^*$ 处为零向量（$\nabla g_i(x^*)=0$）时，该约束在 $x^*$ 附近的行为是“平坦”的一阶近似失效。此时令 $\lambda_0=0$、该 $\lambda_i\neq 0$、其余乘子为零，FJ 方程自动成立——\emph{无论 $x^*$ 对目标函数来说是好是坏}。

\begin{notebox}
\textbf{一句话：} FJ 条件“太松了”——在没有额外假设的情况下，它可能把一些跟目标函数毫无关系的点也标记出来。这正是引入约束规格的动机。
\end{notebox}

动机的关键页面。

\subsection{6.4 KKT 条件——FJ 的“补丁版”}

KKT 条件在 FJ 的基础上\textbf{加上约束规格}，强制 $\lambda_0 \neq 0$，然后归一化 $\lambda_0=1$。这样目标函数的梯度永远不会被“消掉”。

PPT第7.19和7.20页给出了 KKT 条件的矩阵形式和详细表述，第7.27页展示了用 Lagrange 函数的紧凑写法。

\medskip
\refslide{7最优性条件}{19}{7.11}
\medskip
\refslide{7最优性条件}{20}{7.12}
\medskip

\begin{defbox}
\textbf{定理 6.3（KKT 条件，Karush--Kuhn--Tucker）}

设 $x^*$ 是局部极小点，且 $x^*$ 处某约束规格成立。则存在乘子 $\lambda_i\ge 0$（对 $g_i$）和 $\mu_j$（对 $h_j$），满足：

{\footnotesize
\begin{enumerate}
  \item $\displaystyle\nabla f(x^*) + \sum_{i=1}^m \lambda_i \nabla g_i(x^*) + \sum_{j=1}^p \mu_j \nabla h_j(x^*) = 0$
  \item $g_i(x^*) \le 0,\; h_j(x^*) = 0$ （原始可行性）
  \item $\lambda_i \ge 0$ （对偶可行性）
  \item $\lambda_i g_i(x^*) = 0$ （互补松弛条件）
\end{enumerate}
}

\vspace{2pt}
定义 Lagrange 函数 $L(x,\lambda,\mu)=f(x)+\sum\lambda_i g_i(x)+\sum\mu_j h_j(x)$，
则 (1) 即 $\nabla_x L = 0$，(4) 即 $\lambda_i g_i = 0$。
\end{defbox}

\medskip
\refslide{7最优性条件}{27}{7.19}
\medskip

\subsubsection{约束规格到底解决什么问题？}

\textbf{约束规格（Constraint Qualification, CQ）} 保证了一件事：\emph{在 $x^*$ 处，约束梯度所描述的局部几何与实际的可行域局部几何一致}。

如果没有 CQ，可能出现这种情况：从一阶 Taylor 展开看，某个方向 $d$ 满足 $\nabla g_i(x^*)^T d \le 0$（看似可行），但实际上不存在沿该方向的可行弧。此时基于线性化的 KKT 条件可能出错——要么漏掉真正的极小点，要么把不可行的方向当作可行的。

\textbf{LICQ 的直观意思：} 积极约束的梯度向量集合线性无关。

用平面几何比喻：每个积极约束在 $x^*$ 处定义了一条曲线（或曲面），它的梯度 $\nabla g_i(x^*)$ 是\emph{法向量}，垂直于这条曲线。LICQ 要求这些法向量互不“共线/共面”——没有哪个法向量可以写成其他法向量的线性组合。

\begin{itemize}
  \item 如果 LICQ 成立，那么每个约束在 $x^*$ 附近都是“独立的”（去掉任何一个都会改变可行域的形状）。
  \item 如果 LICQ 失败（如 $\nabla g_i(x^*)=0$），意味着该约束在 $x^*$ 附近至少二阶平坦——一阶信息不足以描述它。
\end{itemize}

\begin{defbox}
\textbf{常见约束规格一览}

\medskip
\begingroup
\footnotesize
\setlength{\tabcolsep}{1.5pt}
\begin{tabular}{@{}p{15mm}@{}p{72mm}@{}}
\toprule
\textbf{LICQ} & 积极约束的梯度线性无关。最常用，条件最强但验证简单。\\
\addlinespace
\textbf{MFCQ} & 积极约束梯度正线性相关，比LICQ稍弱。\\
\addlinespace
\textbf{Slater} & 对凸问题：存在严格可行点使 $g_i(x)<0$。验证方便。\\
\addlinespace
\textbf{线性约束} & 若所有约束均为线性，则CQ自动成立，无需验证。\\
\bottomrule
\end{tabular}
\endgroup
\end{defbox}

\subsection{6.5 自编例题：完整 KKT 求解过程}

\begin{notebox}
\textbf{【自编例题2】混合约束的 KKT 完整演练}

$\min\ x_1^2 + x_2^2$，s.t.\ $x_1^2 + x_2^2 \le 1$（单位圆盘内），$x_1 + x_2 = 0.5$（直线上）。

几何直觉：直线穿过单位圆内部。最近原点的垂足 $x_1=x_2=0.25$ 在圆内（$0.125<1$），不等式约束应不活跃。
\end{notebox}

  \draw[dashed, thin, black!60] (0,0) -- (0.25,0.25);

  \filldraw[black] (0,0) circle (1.2pt) node[below left] {$O$};

  \node at (0.6,-1.0) {$\lambda=0$（$g(x)$不活跃）};
\end{tikzpicture}

\medskip
\textbf{图 6.2：KKT条件示例——不等式约束非活跃}
\end{center}

\smallskip
\textbf{第1步：写 Lagrange 函数。}
记 $g(x)=x_1^2+x_2^2-1\le 0$，$h(x)=x_1+x_2-0.5=0$。
\[
L(x,\lambda,\mu) = x_1^2+x_2^2 + \lambda(x_1^2+x_2^2-1) + \mu(x_1+x_2-0.5)
\]
其中 $\lambda\ge 0$（不等式乘子非负），$\mu$ 无约束（等式乘子）。

\smallskip
\textbf{第2步：写 KKT 方程组。}
\begin{align}
\frac{\partial L}{\partial x_1} &= 2x_1 + 2\lambda x_1 + \mu = 0 \label{kkt-a}\\
\frac{\partial L}{\partial x_2} &= 2x_2 + 2\lambda x_2 + \mu = 0 \label{kkt-b}\\
\lambda &\ge 0,\quad \lambda(x_1^2+x_2^2-1) = 0 \label{kkt-c}\\
x_1^2+x_2^2 &\le 1,\quad x_1+x_2 = 0.5 \label{kkt-d}
\end{align}

\smallskip
\textbf{第3步：分情况讨论（互补松弛 $\lambda$ 的两种情况）。}

\textbf{情形 1：$\lambda = 0$（不等式不活跃，最优解在圆内部）。}
由 KKT 方程：$2x_1+\mu=0,\; 2x_2+\mu=0$，故 $x_1=x_2=-\mu/2$。
代入 $x_1+x_2=0.5$：$-\mu=0.5$，得 $\mu=-0.5$，$x_1=x_2=0.25$。
验证：$0.25^2+0.25^2=0.125\le 1 \checkmark$，$\lambda=0\ge 0 \checkmark$。
\textbf{候选点：} $x^*=(0.25,0.25)^T$，$f^*=0.125$，$\lambda^*=0$，$\mu^*=-0.5$。

\textbf{情形 2：$\lambda > 0$（不等式活跃，最优解在圆周上）。}
由互补松弛 $x_1^2+x_2^2 = 1$。
由 KKT 方程 (\ref{kkt-a})(\ref{kkt-b}) 相减得 $2(1+\lambda)(x_1-x_2)=0$。
因 $\lambda>0$ 故 $1+\lambda>0$，得 $x_1=x_2$。
代入 $x_1+x_2=0.5$：$x_1=x_2=0.25$。但 $0.25^2+0.25^2=0.125 \neq 1$，矛盾！
\textbf{情形 2 无解。}

\smallskip
\textbf{第4步：结论。}
唯一 KKT 点：$x^*=(0.25,0.25)^T$，$\lambda^*=0$，$\mu^*=-0.5$，$f^*=0.125$。
不等式约束在该点不活跃（$\lambda=0$），与几何直觉一致。

\begin{notebox}
\textbf{这个例题教了什么？}
\begin{enumerate}
  \item KKT 求解的标准流程：写 Lagrange 函数 $\to$ 写 KKT 方程组 $\to$ 分情形讨论 $\to$ 逐一验证。
  \item 互补松弛 $\lambda g(x)=0$ 本质上是“开关”——$\lambda=0$ 时约束关闭（不活跃），$\lambda>0$ 时约束打开（活跃在边界）。
  \item 若无不等式约束，直接 Lagrange 得 $(0.25,0.25)$；加上后仍在圆内，约束冗余——KKT 正确识别（$\lambda=0$）。
\end{enumerate}
\end{notebox}

\subsection{6.7 真题解析：第六题(1)(2)——FJ与KKT的经典考试题}

\begin{notebox}
\textbf{【2025 真题·第六题（节选）】}
\[
\min\ x_1^2 + x_2^2,\quad \text{s.t.}\ (x_1-1)^3 - x_2^2 = 0
\]
(1) 用 KKT 条件求最优解，证明不存在可行点满足 KKT 条件；(2) 用 FJ 条件找出所有满足 FJ 条件的可行点。

\textbf{命题意图：} 约束梯度在最优解处退化（$\nabla g=0$），体会 FJ 与 KKT 的本质差异。
\end{notebox}

\subsubsection{6.7.1 问题分析}

目标：$f(x)=x_1^2+x_2^2$，$\nabla f(x)=(2x_1,\;2x_2)^T$。\\
约束（等式）：$g(x)=(x_1-1)^3-x_2^2=0$，$\nabla g(x)=(3(x_1-1)^2,\;-2x_2)^T$。

\textbf{注意 $\nabla g$ 的退化点：} 当 $x_1=1,x_2=0$ 时，$\nabla g(1,0)=(0,0)^T=0$——LICQ 不成立。\\[2pt]
同时 $(1,0)$ 满足约束 $(0)^3-0^2=0$，它是可行点且恰为目标极小点（$f=1$ 是可行集上的下确界）。

\subsubsection{6.7.2 (1) 尝试 KKT——不存在}

KKT 条件（归一化 $\lambda_0=1$，乘子 $\mu$ 无约束）：
\[
\begin{pmatrix} 2x_1 \\ 2x_2 \end{pmatrix}
+ \mu \begin{pmatrix} 3(x_1-1)^2 \\ -2x_2 \end{pmatrix}
= \begin{pmatrix} 0 \\ 0 \end{pmatrix}
\]
且 $g(x)=(x_1-1)^3-x_2^2=0$。

即方程组：
\begin{equation}
2x_1 + 3\mu(x_1-1)^2 = 0 \label{eq:kkt-1}
\end{equation}
\begin{equation}
2x_2 - 2\mu x_2 = 0 \;\Rightarrow\; 2x_2(1-\mu) = 0 \label{eq:kkt-2}
\end{equation}
\begin{equation}
(x_1-1)^3 - x_2^2 = 0 \label{eq:kkt-3}
\end{equation}
由 (\ref{eq:kkt-2})：$\mu=1$ 或 $x_2=0$。

\smallskip
\textit{情形 a ($\mu=1$)：} 代入 (\ref{eq:kkt-1})：$2x_1+3(x_1-1)^2=0$
$\Rightarrow 3x_1^2-4x_1+3=0$。$\Delta=16-36=-20<0$，无实根。

\textit{情形 b ($x_2=0$)：} 由 (\ref{eq:kkt-3}) 得 $x_1=1$。代入 (\ref{eq:kkt-1})：$2+0=2\neq 0$，矛盾。

\smallskip
\textbf{KKT 结论：不存在满足 KKT 条件的可行点。} $\square$

\subsubsection{6.7.3 (2) FJ 条件求解}

FJ 方程（引入 $\lambda_0\ge0$，乘子 $\mu$ 无约束，$(\lambda_0,\mu)\neq(0,0)$）：
\[
\lambda_0 \begin{pmatrix} 2x_1 \\ 2x_2 \end{pmatrix}
+ \mu \begin{pmatrix} 3(x_1-1)^2 \\ -2x_2 \end{pmatrix}
= \begin{pmatrix} 0 \\ 0 \end{pmatrix}
\]
且 $g(x)=(x_1-1)^3-x_2^2=0$。

\smallskip
\textbf{情形 1：$\lambda_0 = 0$（退化情形）。}
方程化为 $\mu\cdot 3(x_1-1)^2 = 0$，$\mu\cdot(-2x_2)=0$。
因 $(\lambda_0,\mu)\neq(0,0)$，$\mu\neq 0$，故 $x_1=1,\;x_2=0$。
代入约束 $(1-1)^3-0^2=0\;\checkmark$。
\textbf{得 FJ 点：} $(1,0)^T$。乘子 $(\lambda_0,\mu)=(0,\mu)$，$\mu$ 任意非零。

\smallskip
\textbf{情形 2：$\lambda_0 \neq 0$（非退化情形，即 KKT 情形）。}
已在上节证得无解。

\smallskip
\textbf{FJ 结论：} 唯一 FJ 可行点 $(1,0)^T$，来自退化情形 $\lambda_0=0$。

\subsubsection{6.7.4 深层解读：这道题到底在考什么？}

\begin{notebox}
\textbf{考试核心信息：}

\begin{enumerate}
  \item \textbf{FJ 找到了全局极小点 $(1,0)$，但通过退化情形 $\lambda_0=0$。} 目标函数信息完全未用——只是凑巧 $\nabla g(1,0)=0$ 满足方程。FJ 在\emph{任何约束梯度为零的可行点}都可能触发，无论该点对目标函数好坏。

  \item \textbf{KKT “漏掉”了真正的极小点。} 不是 KKT 的错——在 $(1,0)$ 处 $\nabla g=0$，LICQ 不成立，KKT 的前提被破坏。KKT 是在“约束足够好”的前提下给出必要条件。

  \item \textbf{几何：} 约束 $(x_1-1)^3=x_2^2$（即 $x_2=\pm(x_1-1)^{3/2}$，$x_1\ge 1$）在 $(1,0)$ 处有\emph{尖点}。一阶信息无法刻画曲线局部走向——FJ 在此退化是\emph{非光滑几何}的必然结果。

  \item \textbf{考试答题模板：}
  \begin{itemize}
    \item FJ 题：分 $\lambda_0=0$ 与 $\lambda_0\neq 0$ 讨论，$\lambda_0=0$ 时用“不全为零”确定候选点
    \item KKT 不存在题：证明 $\lambda_0\neq 0$ 情形无解
    \item 关键：检查 $\nabla g$ 是否退化（零向量）——退化点是考 FJ vs KKT 差异的经典载体
  \end{itemize}
\end{enumerate}
\end{notebox}
